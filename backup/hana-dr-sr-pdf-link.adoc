:hardbreaks:
:allow-uri-read: 
:nofooter:

Questo documento fornisce una panoramica delle opzioni di protezione per il disaster recovery per SAP HANA. Include informazioni dettagliate sulla configurazione e una descrizione del caso d'uso di una soluzione di disaster recovery a tre siti basata sulla replica dello storage NetApp SnapMirror sincrona e asincrona. La soluzione descritta utilizza NetApp SnapCenter con il plug-in SAP HANA per gestire la coerenza del database.
