= Backup e ripristino SAP HANA utilizzando i sistemi di storage NetApp e il software Commvault
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
Questo documento descrive la progettazione di una soluzione NetApp e Commvault per SAP HANA, che include la tecnologia di gestione degli snapshot Commvault IntelliSnap e la tecnologia Snapshot NetApp. La soluzione si basa sullo storage NetApp e sulla suite di protezione dati Commvault.

Autori: Marco Schoen, NetApp; Dr. Tristan DAUDE, CommVault Systems
